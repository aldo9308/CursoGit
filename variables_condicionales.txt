Declaracion de variable en Laravel

Para hacer una declaracion de una variable en laravel se 
utiliza las llaves dobles, o dos signos de exclamacion,
{{$nombre}} o {!!$nombre!!}.
o como una declaracion normal en php
<?=$edad?>

Include 


<h1 style="background:yellow"> Esto es una cabezera</h1>

Se tiene que colocar la instruccion @include y señalar el elemento
a incluir.

@include('contacto.cabecera')

se le puede colocar arrays o informacion adentro, por ejemplo:

<h1 style="background:yellow"> Esto es una cabezera</h1>
@if(isset($nombre))
<h2>{{$nombre}}</h2>
@endif

En la pagina principal se coloca la siguiente instruccion para
que mande a llamar a la variable nombre

@include('contacto.cabecera', array('nombre' => $nombre))




middleware es un pequeño metodo que se ejecuta antes o despues
de ejecutar una accion de nuestro controlador.
